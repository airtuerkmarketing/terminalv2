## The one deviation worth knowing

**D-089 was scoped to 5 of the 8 helpers in `SECDEF_REVOKE_TEST_PLAN.md`.** A `pg_policies`
pre-flight found that `is_admin`, `can_access_document_folder`, and `can_see_document_folder`
are referenced by the **`{public}`** policies `document_folders_select` / `document_files_select`,
which serve `is_public` Document-Library files to **anonymous** visitors by design
(`src/lib/documents.ts`). The plan's blanket 8-helper revoke would have made **every anon library
read return `permission denied for function …` (HTTP 500)** — and its admin/super/auth-only test
matrix would not have caught it. The 3 are kept anon-executable; the other 5 (referenced only by
`{authenticated}` policies, and not called by any app `.rpc()`) are revoked. Real anon REST
verification: `document_folders` 200/2 rows, `presentation_folders` 200/0, `rpc/is_super_admin`
401, `rpc/is_admin` 200.
